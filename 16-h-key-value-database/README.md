## Features

- **SET**: Store a value associated with a key.
- **GET**: Retrieve the value associated with a key.
- **DEL**: Delete a key-value pair.
- **EXIT**: Exit the program.
- **Thread Safety**: Uses `sync.RWMutex` to ensure safe concurrent access.
